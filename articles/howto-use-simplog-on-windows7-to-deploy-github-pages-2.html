Git和GitHub动手玩

上一篇文章简单介绍了一些预备知识（是的我知道很枯燥-_-）。 本文将会介绍实际的使用方法。最好学习方法就是自己动手，丰衣足食。

  1. 首先你需要先在GitHub上注册属于自己的用户名。这个过程就不写了，如果你这都不会，老天也帮不了你了。

  2. 点击New Repository新建一个Repository。

  3. 填写Repository name，如下图，如果你的用户名叫

    yourusername
    那么你的Repository Name必须填写为
    yourusername.github.com
    这样GitHub就会自动根据这个特殊的代码仓库里的文件生成网站了。

  4. 点击msysgit下载Git安装包，然后在你的Windows7上安装Git。
